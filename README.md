# MtaStats
A Rails 5.0 and Angular 4 project that uses the MTA's developer API to calculate subway statistics. 
1) Clone project
2) Ensure [rvm](https://rvm.io/rvm/install) is installed
3) `rvm install 2.4.1`
4) `rvm use 2.4.1`
5) `gem install bundler --no-doc --no-ri`
6) `cd MtaStats`
7) `bundle install`
